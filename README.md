## Getting Started

#### Clone the repository

```sh
git clone git@github.com:nicoayamen/dicer.git
```


#### Install dependencies for both frontend and backend
```
cd dicer

cd frontend
npm install

cd ../backend
npm install
```


####  Running the Development Servers
Backend

```cd backend
npm run local
```

Frontend
```cd frontend
npm start
```
